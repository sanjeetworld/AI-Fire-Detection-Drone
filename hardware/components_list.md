# 🚁 AI Fire Detection Drone – Components List

This project integrates drone hardware, AI-based fire detection, and IoT communication modules for real-time disaster response.

---

## 🛩️ Drone Hardware Components

- 4× **1000KV BLDC Motors**  
- 4× **30A Electronic Speed Controllers (ESC)**  
- 4× **Propellers (Blades for motors)**  
- **KK2 Flight Controller Board**  
- **Lithium-ion Battery (22000 mAh)**  
- **Buzzer (Low Voltage Alarm)**  

---

## 🎮 Controller System

- **FlySky CT6B Transmitter & Receiver** *(or latest FlySky model recommended)*  

---

## 🔥 Fire Detection & AI Module

- **Flame Sensor (Fire Detection)**  
- **Smoke Sensor (MQ-2 or similar)**  
- **Temperature Sensor**  
- **ESP32 Microcontroller**  
- **Servo Motor (for extinguisher mechanism)**  

---

## 📡 Communication & Tracking System

- **GSM Module (SIM800L / SIM900)**  
- **GPS Module (NEO-6M or similar)**  

---

## ⚙️ Additional Components (Optional but Recommended)

- Jumper Wires  
- Breadboard / PCB  
- Power Distribution Board  
- Frame (Drone body)  
- Camera Module (for real-time monitoring)  

---

## 📌 Notes

- Ensure all components are compatible in terms of voltage and current ratings.  
- Use proper battery management for safety.  
- Calibrate ESC and flight controller before flight.  
- Test fire detection modules separately before integration.
